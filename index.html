Code Retreat Groningen

Eerstvolgende retreat: 15 februari 2014

Type: Legacy Code Retreat

Op zaterdag 15 februari organiseren wij een legacy code retreat in Groningen. Een code retreat is een (gratis) dag lang je programmeerskills aanscherpen. We werken niet aan een specifieke taal, technologie of framework, maar richten ons de hele dag lang op de kern: programmeren.

Deze dag organiseren we bij Ordina in Groningen, en duurt van 9:00 tot 17:00. De lunch is inclusief, en de dag is volledig gratis. We vragen je om een laptop mee te nemen met ingerichte ontwikkelomgeving naar keuze. Als dat echt niet lukt is het niet perse een probleem, maar meld het dan wel even.

Wat is een legacy code retreat?

Gedurende de dag zul je pair programmen met allerlei mensen, telkens andere pairs, en steeds sessies van circa 45 minuten. Je krijgt van ons een programma wat onleesbaar en onnavolgbaar is. Aan jullie de taak dit programma dusdanig te refactoren, dat je er trots op kan zijn. Om je hierin te begeleiden krijg je elke keer specifieke opdrachten, elk gericht op bepaalde refactortechnieken.

Door zo beperkt te werken ondervind je welke technieken in welke gevallen dat goede aanpakken zijn, en wanneer je beter een andere techniek kan toepassen.

De doelgroep is programmeurs. Eigenlijk typisch de programmeurs die in hun baan niet altijd de tijd vinden om oneindig door te perfectioneren. Iedereen ontwikkelt in de taal en IDE/editor waar hij of zij zich het beste in kan vinden (Ruby, Java C#, …). Het gaat niet om de programmeertaal maar om de aanpak!
